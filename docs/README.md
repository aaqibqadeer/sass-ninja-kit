# Docs

Documentation index for the ninjakit boilerplate. One line per subfolder.

- **knowledge-base/** — shared agent memory; **read `current-state.md` first**.
  Also `decisions.md` (why non-obvious choices were made) and `glossary.md`.
- **architecture/** — living reference docs: `overview.md` (philosophy),
  `feature-flags.md`, `data-layer.md` (adapter pattern), `components.md`
  (reusable component catalog), `theming.md` (theme tokens).
- **guides/** — human-readable setup/usage walkthroughs; start with
  `getting-started.md`. Per-feature guides added as features land.
- **prompts/** — standalone, copy-pasteable prompts for common tasks
  _(populated in later phases)_.
- **llm-context/** — source material for `CLAUDE.md` / `.cursorrules`
  _(populated in later phases)_.
- **legal-templates/** — boilerplate legal docs (terms, privacy, etc.)
  _(populated in later phases)_.
