# Docs

Documentation index for the sass-ninja-kit boilerplate. One line per subfolder;
filled in as phases progress.

- **architecture/** — system design notes; start with `overview.md` (flag-driven + adapter philosophy).
- **guides/** — how-to guides for using and extending the boilerplate _(populated in later phases)_.
- **prompts/** — reusable prompts for driving development with an AI assistant _(populated in later phases)_.
- **llm-context/** — condensed context files to feed an LLM about this codebase _(populated in later phases)_.
- **legal-templates/** — boilerplate legal docs (terms, privacy, etc.) _(populated in later phases)_.
